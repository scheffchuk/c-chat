# C Chat: マルチモデル AI チャットアプリケーション

## C Chat とは？

複数の AI アシスタントを 1 つのインターフェースから利用できる Web アプリケーション。異なる AI モデルの応答を比較し、簡単に切り替えながら会話を管理できます。

## 対象ユーザー

- **学生**: AI モデルの仕組みを学ぶ人
- **開発者**: AI アプリを構築する人
- **研究者**: AI モデルの性能を比較する人
- **すべての人**: 複数の AI アシスタントを効率的に利用したい人

## なぜ C Chat なのか？

既存ツールの問題（高コスト、サービス縛り、使いにくさ）を解決：

- 無料のオープンプラットフォーム
- 複数 AI モデルへの一元アクセス
- 高速で応答性の高い UI
- プライバシー重視

## できること

### コア機能（現在利用可能）

- **AI チャット**: 複数モデルと会話
- **応答比較**: 異なるモデルの回答を比較
- **自動保存**: 全チャットを自動保存
- **認証**: 安全なアカウントシステム
- **テーマ**: ダーク/ライトモード切替
- **多様な入力**: テキスト、音声、ファイル添付
- **リアルタイム**: 応答をストリーミング表示
- **アーティファクト**: インタラクティブなコンテンツ（コードサンドボックス、マップ、視覚化など）で会話を拡張。テキストだけでなく、チャット内で実行可能なツールや視覚化を作成可能。

### 近日公開

- **チャット履歴**: 過去の会話を閲覧・検索
- **メッセージアクション**: アップボート、会話分岐
- **設定**: カスタマイズ可能な設定
- **公開/非公開**: チャットの共有制御

## 使い方

1. **サインイン**: アカウント作成またはログイン
2. **チャット開始**: 「New Chat」をクリック
3. **モデル選択**: 使用する AI モデルを選択
4. **質問**: テキスト入力、音声、またはファイル添付
5. **応答確認**: リアルタイムで応答を確認
6. **自動保存**: 会話は自動保存

## 特徴

- **マルチモデル**: 簡単にモデル切替
- **リアルタイム**: 応答をストリーミング表示
- **プライバシー重視**: 安全なデータ保存
- **モダン UI**: デスクトップ・モバイル対応
- **オープン**: 単一プロバイダーに縛られない

## はじめに

必要なもの：

- Web ブラウザ（Chrome、Firefox、Safari、Edge）
- インターネット接続
- アカウント（無料）

Web サイトにアクセスしてサインインし、チャットを開始。
